SOLUTION 1:
1. download "Blackd Proxy Alternative Installer - Zipped MSI file"
2. unzip it
3. run Blackd Proxy.msi to install it.
4. after installation just run blackd proxy (Do NOT run as admin)

SOLUTION 2:
if you have UAC mode ENABLED
1. download "Blackd Proxy Installer - Complete installation" from http://www.blackdtools.com/freedownloads.php
2. right click over it and choose "run as admin" (or something like that depending your language) This should install everything.
3. after installation, just run blackd proxy (this time you should just run it. Do NOT run it as admin.)

SOLUTION 3:
if you have UAC mode DISABLED
(if solution 2 doesn't work)

Credits to Elvis Dourado who contacted me by email with his solution:

1. Download and save in windows\system32\ ...
dx7vb.dll (http://www.blackdtools.net/directdownload123/dx7vb.dll)
MSWINSCK.OCX (http://www.blackdtools.net/directdownload123/MSWINSCK.OCX)
msflxgrd.ocx (http://www.blackdtools.net/directdownload123/msflxgrd.ocx)
(you can click in the 3 names above to download them directly from my host, or just download them from any other web if you feel more safe that way)

2. search and execute CMD.EXE as administrator. A black window will popup. There you should type:
regsvr32 /s mswinsck.ocx
regsvr32 /s dx7vb.dll
regsvr32 /u /s msflxgrd.ocx
regsvr32 /u /s mswinsck.ocx
regsvr32 /u /s dx7vb.dll
regsvr32 dx7vb.dll

(might be redudant, but for me it somehow it worked after executing exactly all that in that order)

SUCESFULLY TESTED BOTH SOLUTIONS MYSELF, USING VISTA ULTIMATE 32 bits

SOLUTION 5:
if you have windows 7 then go to user options in control panel and drop the security level to minimum. Then all required dlls and ocxs should register at installation without problems

MISC NOTES:

- Avoid to install Blackd Proxy in strange locations. The default installation path should be fine ( C:\program files\blackd tools\blackd proxy ) You can also use C:\blackdproxy or anything else, but please AVOID desktop, my documents, temp, windows, or such special/protected folders.

- UAC mode should be ON
