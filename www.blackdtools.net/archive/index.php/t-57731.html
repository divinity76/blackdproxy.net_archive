Now there is a new menu at start. It will allow you to set all your OT server config correctly in the same place, in a fast way:

7385

Just select your Tibia version, and in case of OT servers just fill:
- tibia path
- tibia map folder
- ot server ip (or domain)
- ot server port
You don't need anything else now (besides having a Tibia client designed for your ot server)

Blackd Proxy will load .dat from your desired ot client folder. It doesnt care if your ot client have a custom tibia.dat name like something.dat Blackd Proxy will now read it there.


Note: you still have to modify some tiles if your server modified the default tiles for important things. Placing a custom config.override.ini in your configxxx folder is the best way. That way you can share that config file with your friends.
Example, for tibianic.org (port 7272, version 7.72) , just make this config772\config.override.ini


[tileIDs]
tileID_firstFoodTileID=6B 0A
tileID_lastFoodTileID=6B 0A
tileID_Blank=D4 08
tileID_Backpack=D0 07
tileID_UH=E1 08
