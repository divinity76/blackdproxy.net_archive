Para usar Blackd Proxy en cualquier OT server necesitarás saber:
- la versión de ese OT server
- la ip del OT server
- si tiene cliente propio modificado o usa un cliente estandar

1. Instala el cliente de Tibia que necesites en cualquier carpeta, por ejemplo C:\TibiaOT

2. Instala la última versión de Blackd Proxy, la cual permite configurar facilmente OT servers.

3. Ejecuta Blackd Proxy. En el primer arranque que hace un Blackd Proxy recien instalado se te mostrará una pantalla de configuración donde deberás introducir:
- la versión de ese OT server
- la carpeta donde has instalado tu cliente de Tibia especial para ese OT server
- la carpeta donde estan los mapas (normalmente funciona con la que calcula Blackd Proxy automáticamente)
- la ip del OT server
- el puerto del OT server (normalmente funciona con el que te indica Blackd Proxy por defecto)

4. Ejecuta tu cliente de Tibia especial, haz login y prueba a usar alguna cosa de Blackd Proxy para comprobar que todo fue bien.
