This tutorial will teach you how to compile BlackD Proxy from source.

First, you need to disable / "Never Notify" on User Account Control, because it doesn't
play well with Visual Basic 6. learn how to do that here: http://www.howtogeek.com/howto/windows-vista/disable-user-account-control-uac-the-easy-way-on-windows-vista/

then, you'll need a Torrent client, like Transmission-qt: http://goo.gl/XjDC9d

In your torrent client, click "Open URI" or "Open URL" or "Open Magnet Link" (it has slightly different names in different clients),
and download the "magnet link" found inside this URL: http://paste.debian.net/plain/366563
(I can't inline the url because the forum is retarded and mess with "%" characters :( )

http://i.imgur.com/lSS58Tn.png
http://i.imgur.com/Kc9qUFV.png
http://i.imgur.com/Zm13V2o.png
when that has finished,
if you run Windows 7 or older, you will need a ISO mounter,
like Virtual CloneDrive: http://static.slysoft.com/SetupVirtualCloneDrive.exe
If you get this warning during setup, say "Install"
http://i.imgur.com/9tSzpFx.png

When the torrent file has downloaded, go to the "visual basic 6" folder
http://i.imgur.com/4WaCxAE.png
then right-click on "MS VS 6.0.iso", and click "Mount" button
http://i.imgur.com/zEXynfM.png
now go to Computer, you should now see a new drive called "MS VS 6.0"
http://i.imgur.com/8FhO9bw.png

right-click on it and press "Install or run program from your media"
http://i.imgur.com/yGHNaIt.png

now you might get this warning:
http://i.imgur.com/YvSMumM.png
just press "Run program"

this window will pop up:
http://i.imgur.com/NmMXGKG.png

press Next

http://i.imgur.com/1OloPrj.png

press "I Accept", then press Next

http://i.imgur.com/HvdAuZy.png

in Product Number, write
450 - 6623317

then press Next

http://i.imgur.com/mD4FSMw.png

press Next again

http://i.imgur.com/NLIzuFV.png
if you get this warning, press Run Program

http://i.imgur.com/qDt2Jd0.png
if you get this warning, press Run Program.

then you will get a warning "will now restart", press OK, then restart the computer.

you may now get some error about "could not find E:\foo.bar", press OK and ignore it.

http://i.imgur.com/N8yvdBs.png
now you need to "mount the ISO" again, same like last time.

http://i.imgur.com/Y0045dk.png
then go to Computer, and press "Install or run program from media" again,

http://i.imgur.com/Bq90LAs.png
you may get this warning again, press "Run program"

http://i.imgur.com/BqBbOem.png

press Next

http://i.imgur.com/4TzD76R.png

press "I accept", then "Next"

http://i.imgur.com/ZUrlFNC.png

in Product Number, write
450 - 6623317

then press Next

http://i.imgur.com/klAefSo.png

press "Custom", then press "Next"

http://i.imgur.com/UfelXl0.png

write
"C:\vb6\Microsoft Visual Studio\Common" in choose folder
then press Next

http://i.imgur.com/maJswZo.png
if you get this warning, press "Run program"

http://i.imgur.com/xr3Dy5d.png

press "Continue"

http://i.imgur.com/alccF2W.png

press "OK"

http://i.imgur.com/uMbpx8f.png

uncheck "Visual C++" and "Visual FoxPro" and "Visual SourceSafe", but not
anything else. then press Continue.

http://i.imgur.com/If758V6.png

press "Restart windows",
then wait until windows restarts (or alternatively, restart it yourself)

http://i.imgur.com/Z9Zs7nK.png
if you get this error, just press OK
http://i.imgur.com/I39Z2Da.png

Now download "Service Pack 6 for Visual Basic 6.0" at https://www.microsoft.com/en-us/download/details.aspx?id=5721
http://i.imgur.com/I39Z2Da.png
http://i.imgur.com/kojCOi0.png

http://i.imgur.com/X1lg9ne.png

start it, and write C:\vb6\sp6\ and press OK

http://i.imgur.com/uG1grE9.png

press Yes

When it has finished,
http://i.imgur.com/DLeqani.png
press the Windows button + R and write
C:\vb6\sp6\setupsp6.exe
and press OK

http://i.imgur.com/WKY0JsQ.png

press Continue


http://i.imgur.com/mJVhA9M.png

press "I Agree"

http://i.imgur.com/FBTF3qK.png

press OK

http://i.imgur.com/kCMYCVj.png
then go to https://github.com/blackdtools/Blackd-Proxy-CLASSIC
and press the "Download ZIP" button

(Actually i recommend getting a github.com account and installing "Github For Windows",
but that is out of scope for this tutorial.),
http://i.imgur.com/CmYMNCX.png

open the ZIP file, and drag the folder to desktop (IT is very important that you drag the folder OUTSIDE the zip file)

http://i.imgur.com/nvmzjWw.png

open the new folder, and find "BlackdProxy.vbp", and double click it.

if everything is ok now you should see something like this:
http://i.imgur.com/3zpjelS.png

http://i.imgur.com/mesD2vX.png

... now press "Run" and "Start with Full Compile",

and you should see BlackD Proxy starting.
Congratulations! you have just compiled BlackD Proxy from the source code!
http://i.imgur.com/lTyiIws.png
you can now start playing with the source code :-)

Any questions/problems?
