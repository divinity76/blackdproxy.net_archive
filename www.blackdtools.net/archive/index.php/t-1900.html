classic mode will only do a smart use of UHs found in opened bps
default mode will use opened bps unless no uhs left in opened bps, else it will spam orders to server to get uh'ed from any bp [waste]
war mode will just spam orders to server to get uh'ed from any bp [waste+]

When to use each mode?

spaming uh orders disallow casting any other order, so classic mode might be good for special events like anihilator.

default mode is good for caveboting, it allows you to survive after a kick/relog , as you wont have any bp opened after it.

war mode might be better in wars where you dont care about waste in order to survive to massive attacks.
