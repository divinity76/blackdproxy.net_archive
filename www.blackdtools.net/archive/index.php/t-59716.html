OK - How to find the ip/port of a Tibia OT server:
------------------------------------------------
0. close all browsers, games and anything that might be using a connection. This will make everything easier.
1. execute CMD
2. that will open a black window. There type: netstat
3. login to your ot server just so it display the character list
4. repeat (1) and (2) , then compare the results. The new line will tell you the ip and port of your ot server.
