do something like this into your script

if true $time$ = 09:30:00 goto label XXXX

:XXXX <-- your labe
sayMessage exiva close


-------------------------

variables:

$time$
Returns current time in HH:MM:SS. Only string I believe.


$shorttime$
Returns current time in HH:MM. This might be both string and number.

Just in case anyone needs this is what I did:
- Created conditional event with 100ms "few time ago": if $shorttime$ #string=# 08:50 action: "exiva _exitnow=1"
- Before the gotoscriptline 0 of I added "IfTrue ($_exitnow$#number=#1) Goto $nlineoflabel:logout$".
- After the gotoscriptline 0 of all my cavebot script I added the label "logout".
- Added waitX 30
- Added fastExiva Close
