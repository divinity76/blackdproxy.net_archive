#=== here we will answer, what exactly is Surge and what can it do?

Surge is now a single control script that is needed to allow features that could

not be done with simple variables alone. Surge is designed with fully AFK

ServerSave->ServerSave in mind, with the exception of kicks or PK. The Script

has advanced options and all the basic functionality as other scripts but these

functions have been improved on in big ways.
Surge can be used on its own through rook but will not go to main without

Bloodmoney's WayPoints.{see:.post 3}

3
#=== Basic settings:
fastExiva _world = World
the custom variable _world tells Surge the name of the world you are playing

on. this is important because if you play more then one world Surge has to know

to only team with chars in the same world. if you play the same world on all

accounts you still need to set this because your bot will use this information

to gather more information and if you have the _speak variable ON it needs to

know what world your on.
fastExiva _ServerSave = 3:00
what is the exact time of Server Save? this is so that the bot will go

someplace safe 15min before SS

4
#=== Char & Outfit settings:
fastExiva _vocation = knight
Set the vocation you want to be. if you already have a vocation other then this

it will be bypassed with _voc. Rookers have no vocation so if you are in rook

you are treated as a knight by default.
*accepted input(Knight,Paladin,Druid,Sorcerer) premotions are not accepted to

prevent using scripts on premmy accounts.
fastExiva _weapon = sword
Set the weapon type you want as your main weapon. if you are not a knight this

sets your secondary weapon type. most if not all of rook will use this type.
*accepted input(sword,axe,club,fist) if using fist set _ironman = 5 or more just

to be safe
fastExiva _city = thais
city in main you want to be a citizen of and hunt at. Surge can walk you to

other cities and make you a citizen of them
*accepted input(thais,carlin,abdendriel,venore,kazordoon)

section
5
#=== advanced settings:
fastExiva _PowerSurge = 0 focus on EXP

vs. loot
sets loot off if the char has 200+gp. makes decisions based off of expgain
1=on,0=off,2=auto
fastExiva _IronMan = 0 adjust lvl

requirement by # lvls
basically a handicap. say your bots keep dieing from rats, if u tell the bot it

needs to be +1 lvl to hunt rats then you might last longer in caves.(not the

fastest exp)
0=off,1=+1 lvl(1-99, I wouldn't set it past 20)
fastExiva _TeamPlayer = 0
soon your bots will be able to cummunicate with eachother. this feature will

allow this; say you have 2 bots running at the same time. your bots will tell

eachother were they plan on hunting so that there not both hunting the same

place, or if a bot is getting way more exp then expected other bot will join the

first bot. if a bot is getting lower then expected exp then bot will notify the

first bot and depending on the situation may ask the second to join him or find

a lower lvl hunting spot.
1=on,0=off,2=auto

6
#=== Rook settings:
fastExiva _Rooker = 0 stay in rook


pretty simple, if this feature is on then bot will not go to main and will

continue as a rooker. If this setting is on there will be a warning comment

every 2 min for the first and last lvl in rook.
0=off,1=on
fastExiva _logatlvl = 99 lvl that bot

will log-out if still in rook.if _Rooker = 1 will bypass this
this is the lvl that your bot will log out at. basicaly only used if you want to

logout befor leaving rook but can be used in main just the same. just set the

lvl you wish to logout at(rookB4main=9)
Note:Remember that if _Rooker is set on then this will have no effect until lvl

99 unless changed.
