Ok this is a guide to what the options do, how to customise them etc.

1. This will choose which hand you will make runes in, Make sure you choose the hand which has the weapon in(If 2 hander you will need to move it to make a rune and if one hander we move the weapon so you can still block monsters)

2. This is what character your runemaker will work for so if your handling more than once char with MC then you can set different settings for each character.

3. This will activate the runemaker so check this box once you've set all your options, hands to make runes, spells, mana costs, safe list etc.

4. With this option on it will eat food out of backpack for as long as you have food.

5. Auto logout if danger in any floor will log you out if any monster/player not in your safe list comes into your screen in any floor(useful if you don't want any chance of dieing/being caught by players).

6. Auto logout if danger in current floor, This logout option will only work if a monster/player not in your safe list come into your screen on your floor. You'll logout as soon as they come into screen unless they shoot you with GFB.

7.This will log you out if you run out of runes or soulpoints.

8. Play the ding.wav when a player messages you, (handy for non-afkers when there watching tv or on bed etc).

9. Plays Player.wav when someone not in the safe list comes into screen.

10. Drinks a mana fluid if its under so much mana.

11. The mana amount which triggers option 10 if selected. (Default is 100).

12. If out of runes or soulpoints then cast the spell selected in option 13.

13. The box where you can change what spell to cast.

14. The mana cost of option 13's spell. (Example exura = 25)

15. The rune spell which will be made if you have enough runes/soulpoints.

16. How much mana option 15 costs.

17.How many soulpoints option 15 costs.

18. On danger play sound, If this box is checked it will play if you are under PK attack or GM comes in screen etc.

19. If your not connected to tibia for any reasons logged out, Internet kick etc play danger.wav.

20. Stop alarms, Does what it says if alarm is going it will stop alarm (Quick way to do this is exiva cancel in the game client)

21. The safe list. Here is all the players/monsters which won't sound alarms you need to load the list everytime you start blackd. If you have added any monsters/players you wish to keep then press the save button to save them on the list.

22. This is how you add/delete monsters/players from the list, Just write thier name in the box and then press add to add them or scroll in the list select thier name and press 'Remove sel' to delete them from the list.

That is the main functions of the runemaker hope it's clearer now if you were having troubles!
