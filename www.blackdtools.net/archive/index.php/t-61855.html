Cavebot
New design and less complicated. Old scripts will work here normally.

https://i.imgur.com/FpWpLL4.png

https://i.imgur.com/uR9DyRH.png
Picture from version 1.0


Remember that you need to set Targets and Loot first, and Loop to finish your script.

Desired Action

Attack:
Melee kill the monster of your choice

Spell Kill:
Cast specific spell to specific monster also determine range in sqm


Misc.

Avoid Wave (circle):
Avoid standing in front of monter avoiding waves and beam

Priority:
The monster having the highest priority will be killed first

Loot ID:
Take loot from the dead creature

Items ID (hex):
Will open a window that shows you different item ids


Emplacement

Walk:
Make waypoint to you current position x,y,z

Ladder:
Climb the ladder of the waypoint sqm

Wait:
Char stop for a moment, set value in seconds

say in Default:
Char will say 'message' in default channel

say in NPC:
Char will say 'message' in NPC channel.
eg. hi, travel, yes

Loop (finish script):
Your script will do a Loop, then it will go to script line 0.
Remember that you need to finish you script with this

Follow Waypoints:
Run Cavebot for selected character.


Save and Load setting:

Save:
Save your script or changes to ".txt"
Dont forget to give a name to your script to avoid overwrite

Load:
Load script on selected ".txt"

Edit:
Will open a window where you can paste script code

Refresh:
Reload your script saved on your blackd folder

Set Cavebot speed:
Randomize actions to avoid getting detected (remember that there is no undetectable bot)
