I think the problem happens if you have an outdated version of tibia.dat in the default tibia folder. You probably failed to update the Tibia client correctly or you downloaded a very early version of the new client.

Try deleting the whole Tibia folder, download the tibia installer again from tibia.com and then reinstall it in the default folder.
