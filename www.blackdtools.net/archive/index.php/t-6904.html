Well Lets Begin!

Since vista includes its own partion tool we wont need any special programs (you will if you want to dual boot from xp)

right click on computer and go to manage

http://img293.imageshack.us/img293/4700/managenz7.jpg

now right click on the drive you want to partion and select 'shrink partion'
the space needed is for the xp installation since xp is about 1.5 gb full instalation i would make it between 1.5 and 2.5 gb


now right click the unallocated memory and select new basic partition
Choose NTFS file system this will save formatting before installation you can name it anything you like its usually 'xp'
http://img182.imageshack.us/img182/5262/vistashrinkjb0.jpg


once formatted you should have your partioned drives in a list usually looks like

(C: ) NTFS
Xp (D: ) NTFS

The partion will be where ur xp installation goes.
