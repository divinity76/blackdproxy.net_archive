This is a picture of the trainer menu:

http://i2.photobucket.com/albums/y9/alucarr/bptutorial/trainer881.jpg
picture from Blackd Proxy 8.81

1. This combobox allows you to select what character you want to configure.

2. This picture control allows you to select what fields are pickupable. Note that if you don't mark any with spears then the autopickuper is disabled. (and this is the default mode)

3. Item ID: here enter what you want to autopick. If you enter 00 00 then you will pick any item

4. Destination of items: here enter where the items will be left when you auto pick up them

5. Here should tell how much items you want to carry as much. When you already have that ammount the autopicker won't pick more.

6. Here you can choose to autorefill any slot. This can be usefull to recharge bolts or to autoequip a new ring/amulet when it is gone. Just set the id of the desired item and what is the minimum ammount that should be there before it refill that slot. Then enable the checkbox for the desired slot (you can enable multiple slot refill)

7. Stop attacking target until regen ... X %hp . This will stop attacks until your target have gained X %hp again. This is not compatible with cavebot yet.

8. Dance at 15 minutes autologout warning: this will do... exactly that.

9. Avoid attacking the monster with this ID. Usefull to slime training. First attack the mother slime for a short moment (don't kill it!) then stop attacks and press the button 10. Then enable the checkbox. Now you can load the slimetrainer cavebot. It will kill all slimes except the one you marked (mother)

10. This button gets the internal ID of the last creature you attacked / marked to attack (even if you didnt touch it)
