Hm, do you mean like 'amount' or item 'ID'. Well, there's a way to do this with a trainer though:

http://i.imgur.com/MP4OB.png

Red square: Select tiles you want to pick up from around you. Works as well if you move your character.

Yellow square: Item ID, quite self explanatory. Put '00 00' to pick up any pickupable item, or change to a specific ID like 'D7 0B' which would pick up gold coins.

Green square: Destination, place where you'd want your items to be carried to, usually backpack or hand if you're training and picking up spears or small stones.

Blue square: Max items you can carry, I usually just put 999999999 there.

Orange square: Timer, usually you don't need to set it up, but if you want bot to pick up items faster, then set it to lower values. I don't know how safe this function is, because it picks up items quite fast, so try to have higher values, because normal human being isn't able to pick items at a high speed. Useful for OT servers though, to put timer at 20ms. Items fly like hell with that.

White square: Once you set up everything right, enable trainer and watch items fly!
