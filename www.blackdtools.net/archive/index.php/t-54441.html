Greetings!

This is yet another of my scripts, I've tested it for a while now so it should be fine!

What it does?
This script will hunt in the main floor north of Port Hope depot.
When full capacity it will go to town deposit the money and refill manas/healths to then continue hunting.
It was made for a Knight so it only buys 20 mana potions and 20 health potions. It goes back to depot if at any time it has less than 5 of any.
Make sure your character can handle 2-3 slimes at the same time (hopefully not for long) and terror birds.
If there is no one in the spawn you should get good exp/h. (Haven't been able to test it alone).

This script has been remade for version 8.71 and using the latest BlackdProxy. Please not that when new updates come I will make sure to update and verify that no hex packets have changed to make sure it is safe and I will update the version number. Use at your one risk if you are using a newer Tibia version.

Feedback
All feedback is welcomed, including ways to make it better.
If any errors, they will be fixed as soon as reported.
