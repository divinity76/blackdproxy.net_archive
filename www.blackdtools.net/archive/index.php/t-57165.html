I had the same problem too, because I got my 'Tibias' installed on Local Disk (D: ) like this:

http://shrani.si/f/2U/Ew/2bk5mv2x/tibias.png

You know, some older versions for playing OTs and some recent versions, because when they update Tibia I just download it from tibia.com and install it on Local Disk (D: ) according to client version they release.

Then, BlackD made awesome BlackD Proxy update to make it safer so it always reads .dat file from Tibia folder so I figured it out that I had to install it on Local Disk (C: ) like this:

My Computer -> Local Disk (C: ) -> Program Files -> Tibia -> Tibia.exe

Now it works like 'cute little kitten' :). Maybe this isn't your case and someone else might also post here on how to solve your problem. :)
