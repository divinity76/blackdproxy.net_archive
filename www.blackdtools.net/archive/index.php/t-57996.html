Ok, I fixed some more things that might have been detected in some way. Now it should be a bit more safe.
I also added some new features.

Blackd Proxy 23.8
----------------------------
- Improved safety of cavebot commands dropLootOnGround and putLootOnDepot. Now slower and more random.
- Now the commands dropLootOnGround and putLootOnDepot are compatible with the new loot mode
- New cavebot command: setBot PKwarnings=0
(this will disable all events that happen when you get attacked by something not included in your scripts)
- Runemaker is now random at multiclient level. Now it is very hard to be detected, even with human eyes.
- New exiva command: exiva drop XX XX AA , drops up to AA items of tile ID XX XX under you.
- Example 1: exiva drop D7 0B 01 : drops 1 gold under you = anti push.
- Example 2: exiva drop D7 0B 02 : drops 2 gold under you = improved anti push.
- Example 3: exiva drop D7 0B FF : drops a full stack of gold under you.




And installer now comes with a new simple looter script. This new script only will work in Blackd Proxy 23.8 or higher.
just_looter.txt

Note: the original script at 23.8 have a little mistake and it says setBot PKwarnings=1. The correct one is setBot PKwarnings=0. I will fix it in 23.9
This is the good one:


setBot PKwarnings=0
setBot LootAll=1
setLootDistance 1
gotoScriptLine 0
