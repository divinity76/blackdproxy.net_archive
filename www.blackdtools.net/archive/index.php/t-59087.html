1st make sure that u know how to use commands with sending packets. Make simple script with packets. For example move 1 gp from first sqm on second sqm.

If that will work fine make script with machete.
But remember that place in backpack is important. So if you record action for machete in 1st sqm of your bp you must have machete always in 1st sqm.

And number of opened bp is important too. If you record for 1st opened BP you will always have got machete in 1st opened BP



If u want send packets by game chat just say

exiva > xxxxxx

and when u want use it in cavebot better use
fastexiva > xxxxxx
