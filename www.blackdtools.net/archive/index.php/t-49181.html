This doesn't have to be a common virus/malware that strokes your keys.. it doesn't have to be a REAL Keylogger.. it can be just a simple Account reader which is using the internal bytes from the Client and makes them into strings which gives them your account&password to prevent this from happening..

Here is a checklist:

1. Check task manager for unrecognized processes (an 3rd party option is Security Task Manager which i use sometimes when i wanna know what a process really is)
2. Get yourself an FireWall with High Block on (don't know what the function is called but on maximum security and you will have to approve EVERY process that has to connect to the internet
3. None working? my advice is to save important files (Do not copy Tibia map, Only appfolder which is located in: %appdata% <- Type in run on start menu, it contains configurations and maps)

I hope this is helping you
