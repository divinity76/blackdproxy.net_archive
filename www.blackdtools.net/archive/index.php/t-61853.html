In Classic Blackd Proxy you need to set the proxy manually, its very complicated.

EG with Normal Blackd Proxy

1.) Open BlackD Proxy.
2.) Open Proxy Section.
2.1.) [Uncheck] Change all Tibia clients memory so they can login to proxy
2.2.) [Uncheck] Change character list packets (use proxy for game connection)
2.3.) [Uncheck] Force login server: ~
3.) Close Proxy section.
4.) Open Tibia client.
5.) Change IP of opened client with an IP changer.
6.) Open Proxy section.
6.1.) [Check] Forward to OT server (shadowcores.twifysoft.net, 7171)
6.2.) [Check] Change all Tibia clients memory so they can login to proxy
6.3.) [Check] Change character list packets (use proxy for game connection)
7.) Close Proxy.
8.) Login.
