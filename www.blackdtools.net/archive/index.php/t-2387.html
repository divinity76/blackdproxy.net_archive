This tool allows you to test how much lag you have in your server and let you know what server is the fastest one for you.

The results are better if you increase the number of pings/server (then the average ping obtained is more reliable)


http://i2.photobucket.com/albums/y9/alucarr/blackdtools/tibiaping.jpg

How does it work internally?

it just connect to gameservers and disconnect as soon the connection is established. This is done several times (4 by default) and the ping is the average of the elapsed times.

The code is included there, avaiable for all the public. It is open source, following the generic gnu license, as all the free tools on this site.
