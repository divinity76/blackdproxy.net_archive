Greetings!

I have recently joined the Blackd community and this is my first released script, first of many I hope.

What it does?
This script will take a new character and level it until it has 6200 experience points. It is fully automated and no interaction is needed between the character and the person.
It know takes your character to main and up to level 11!

How does it do it?
This script will make the character hunt for food to make cash in order to buy decent equipment before starting to hunt. After it reaches the specific amount of golden coins it buys all the equipment and then begins hunting in Al Dee's Sewer cave from level 1 through 4. When it reaches level 4 it then goes hunting on the Northern part of the island since it is faster than hunting rats until level 8.

Changes in Version 2
Although it does not hunt them, I have added trolls in the kill list in order to avoid alarms.
Added auto-continuer in case you get kicked or logged out. (See instructions for details).
Removed areas of poison spiders (many characters that were used to test the script died).
You can now select a vocation, weapon and city and it will take you to main.
Experience Points needed to go to the Mainland is now 6000 to avoid issues with becoming level 9 and not being able to grab new items in Destiny Island.
While hunting in the Northern part if the character has less than 90 hitpoints it will go buy food, eat and wait for hitpoints to recover and then resume hunting. (Many testers died when hunting not on peak hours).
Added Wolf Paws to loot.
Better, More efficient food hunting (Will notice it has no capacity left sooner).


What does it do in Main?
From levels 8-11
Goes to depot and deposits all money (Except 50 gps).
If it has 50 golden coins it will go buy a Rope then hunt the Sewers till level 11.
If it doesn't have that amount it will go hunting in the sewers until it has 50 (it will not get trapped in holes). Once it gets 50 golden coins it will go buy the rope for better hunting.
If it at any point it has lost more than 120 hitpoints it will go buy food, eat it, wait to recover and continue hunting.

Note: Before going to main it grabs the equipment from the quest room in Destiny Island. Feel free to change its equipment as you see fit, but only after it reaches the Mainland.

Other Comments

This script was designed so it can hunt in the area where the pigs and sheep are. Do not worry about it getting trapped with the gates as it will not happen.
It was decided that the wolves cave will be avoided since while testing it was found that even characters of level 7 or 8 can die when sorrounded by 4-5 wolves. (For people who want to know, the script was set to only hunt the wolf cave if above level 6 but it was too dangerous).
Some part of the script where taken from Arne's Script (http://blackdtools.net/showthread.php?t=34020), but everthing in the script was tested carefully. Thanks Arne!
It was noted that although it will get experience faster the character will not loot as many golden coins as other scripts where it just hunts rats but the purpose of this script was to get it to the mainland fast, after all 600 gps are easier to get there.
After it is done remember to check the bank because if at any time your character makes more than 500 golden coins it will go deposit it at the bank.
It was decided to make it for free accounts as most people will use this script to level cash-makers.
Tip: The script was successfully tested while using balanced fighting.


Instructions
1. Create a new character.
2. Login in Tibia.
3. Start script.
Note:Do touch anything else; it will do everything for you.
Remember: Uncheck allow reposition at start.

Instructions for Re-running the Script
This are instructions for when you need to re-run the script on a character that for some reason stopped in the middle of the script running (kick, etc.)
1. Open Tibia and Log in.
2. Bring your character to
If you are still in Rookgaard bring him to temple.
If you in the Destiny Island take him to the place where you arrive to the island.
If you are in the Mainland bring him to Carlin Depot and edit the following line by changing the 0 for a 1:

fastExiva __mainland = 0


3. Start the script.
Warning: Do not open the bag/backpack.


Instructions for Choosing a Vocation
1. Open the file.
2. Edit the number in line where it says:

fastExiva __vocation = 1
Put 1 for Knight, 2 for Paladin, 3 for Druid or 4 for Sorcerer
Note that for now it only works for knights, if you select a different vocation it will only take it to the destiny island and log off.


Instructions for Choosing a Weapon
(will only affect if you choose to be a Knight)
1. Open the file.
2. Edit the number in line where it says:

fastExiva __weapon = 1
Put 1 for Axe, 2 for Club, 3 for Sword
