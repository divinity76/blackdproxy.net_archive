You will need Blackd Proxy 9.02 or higher.

Then just do

exiva bomb targetname

This will load/reload the magebomb , overwriting the target with your given name, for all the characters. This will also refresh the timers to the initial retry timers (extended time for retrys)

Note that, the same than all my commands, you should not use " to delimit long names, and caps doesn't matter, so for example, to magebomb Pepe Perez it would be:

exiva bomb pEpE pEREZ

If you only want to load it with current targets or refresh the timers:

exiva bomb
