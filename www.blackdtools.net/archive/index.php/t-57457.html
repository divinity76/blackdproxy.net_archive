Well, here are the newest changes in Blackd Proxy 22.3:



Blackd Proxy 22.3
----------------------------
- Blackd Proxy now avoid doing unnecessary closes of Tibia clients at Tibia.dat updates.
- It now allows you to keep some config values unmodified through updates if you store them in configxxx\config.override.ini or settings.override.ini
- It now avoids tutorial hints popups.
- Fixed a DirectX problem in Windows Vista/Windows 7
- Fixed a bug with private messages


and last hour update:


Blackd Proxy 22.4
----------------------------
- Blackd Proxy will keep executing even if it is unable to read Tibia.dat from Tibia folder. Anyways it still will give a safety warning in that case.
