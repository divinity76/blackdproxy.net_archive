Blackd Proxy 23.7
----------------------------
- Updated to work in Tibia 9.53
- Fixed another detectable thing in the cavebot looter.
- New inverse loot mode: Enabling this mode the cavebot looter will loot everything, except the IDs added with setLoot. Usage: setBot LootAll=1







About the detectable thing I fixed: While I was doing some looting tests I noticed that in some cases my looter tried to eat wrong things in wrong container slots, so I fixed the eater function, adding a pair of security checks to avoid eating not-food items, and to avoid eating food items in wrong slots.
This was probably one of the last things that the automatic detection system was still detecting as cheat.
But if there are still other flaws somewhere then cavebot might be still detectable. I can't know that yet.
