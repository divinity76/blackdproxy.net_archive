You need to do this through exiva > command.
Reference http://www.blackdtools.net/showthread.php?1706-Custom-cheats-with-exiva-gt
You'll have to first to log the right packet data, stand in place that "move 32823,32090,7" leads to, open Proxy menu in bot, make tibia window small so you can see and switch between tibia and proxy windows easily, now set the checkbox in Proxy that says Log Packets, then in tibia cut grass with machete, unselect log packets in proxy.
Now find in logged packets line that your client sent while cutting grass, it will start with GAMECLIENT > or something like this, copy the hex data (just the numbers and letters that look like 52 AA EE DD for example) and in bot replace heremachetescript with exiva > copiedhexdatahere. Make sure if you copied the right data.
IMPORTANT - REMEMBER to DELETE ALL exiva > commands after EVERY TIBIA UPDATE (when you have to download new client version) and just make new ones by logging packets manually again.
