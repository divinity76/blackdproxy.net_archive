I know that there are several things that can be improved.
But answering to your first part: to force the bot to start in your desired line, go to config.ini and set AllowRepositionAtStart=0

Anyways in version 18.0 it will be changed to a cavebot command (config.ini value will be ignored) This will be the new cavebot command:
setBot AllowRepositionAtStart=0

and I will add a new optional command if you want to disable reposition in traps:
setBot AllowRepositionAtTrap=0
