This new mayor update merges CLASSIC and NG into a new powerfull bot that combines the best of both versions and also adds additional features.

- Mayor bot update. This merges versions Classic and NG. It will load as NG by default.
However you still can show Classic menu clicking Old Menu button at the NG menu.
- New Healer, now its just select combo list for mana/heal pots.
- Heal method Hotkeys/Warmode now use runes with closed backpack (7.72-).
- Auto change gold/platinum checkbox.
- MW timer checkbox
- Added Housing step to safety, runemaker step into house if monster/npc/player not in kill/friend list appear on screen
- Added Tibia Title, shows character name on Tibia title.
- Revamped Aimbot.
- Added Persistent, so player can execute exiva or say message every X ms.
- Classname changer on Advanced, so its possible to recognize some of custom tibia client (untested). Classname on .txt does not recognize most common custom classname (ÙbiaClient).
- Revamped CaveBot form, stills with all button you can show/hide advanced button and added Items ID list.
- Added some new exiva comands (exiva sdmax, etc) and hotkeys list inside Hotkeys form.
- Revamped Broadcast, Trainer, Advanced, Proxy forms.

Credits to oclipper (http://www.blackdtools.net/member.php?74798-oclipper)- the original creator of Blackd Proxy NG mod.
