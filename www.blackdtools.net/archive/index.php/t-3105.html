You can use BlackD on OT and RL tibia at the same time. You do like this:

1. Close all tibia clients and blackd
2. Start 1 tibia client
3. Start an IP Changer and change ip
4. Close the IP Changer and start another tibia client
5. Start BlackD
6. Go to Proxy and change to Forward to OT server and enter the ip and port
7. Connect with the first client you opened (the one with changed ip)
8. Now go back to Proxy and select Forward to true server
9. Now connect with the second tibia client you opened
10. NOW IT IS WORKING!! YEEEHA!

EDIT:
If OT is the same version as RL tibia this will work

Hey, well there's a little easiest way to connect ot server with blackd, namely:
1. Run your blackd, open proxy and set your server IP, login port,
2. Run your Tibia clients and Ip changer,
3. Open proxy and deselect options:
" Change all Tibia clients memory so they can login to this proxy "
" Change character list packets ( use proxy for game connection )"
4. Go to IP changer and change IP on your Tibia clients
5. Go back to Proxy and select back said above options.
6. Login to your server and have fun ;)
