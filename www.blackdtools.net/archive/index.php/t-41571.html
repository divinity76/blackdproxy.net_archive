I've just found it on the net :

Make a new conditional in your blackd folder /conds

Paste this code:



$nameofhex-id:{$hex-lastattackedid$}$
#string=#
barbarian skullhunter
0
250
0
exiva _var = $numericalexp:{$_var$}+1$
$hpofhex-id:{$hex-lastattackedid$}$
#number>=#
5
0
250
0
exiva _var = $numericalexp:{$_var$}+1$
$hpofhex-id:{$hex-lastattackedid$}$
#number=#
100
0
250
0
exiva _var = 0
$_var$
#number=#
2
0
750
0
exori gran
1
#number=#
1
0
250
0
exiva _var = 0

Change:
Red - for monster name
Blue - when the character will have to stop to cast the spell (for example when you can kill it in a single hit melee)
Orange - Spell name

Enjoy! :)
