Malmis
03-30-2012, 01:37
Download and install their custom client
Put a copy of the "tibia.dat" file from the custom Tibia client installation directory in your "Blackd Proxy/config772" (rename the old "tibia.dat" or delete it)
Start Blackd Proxy and open the "Advanced" window
Select "Tibia 7.72" in the "Load a config for different Tibia:" drop-down and press "Do it (will close ALL)"
Start Blackd Proxy and open the proxy window
Set the forwarding option to "Forward to OT server"
Enter "tibianic-hr.org" in server IP and "7272" in "login port:"

Done.

Guys if you want a 7.4 bot; classic botter is a nice one.

And if you want to use blackD on this; take the 7.72 modified clients tibia.dat, replace it with the other tibia.dat in config772.

Then go onto 'advanced' on blackd, make it load for the 7.72 tibia client. Then press 'do it all~'

Open blackd again, go on 'proxy', then go onto Forward to otservers, type in the box; tibianic-hr.org. In the port area, type in 7272. Close the blackd then.

Open the tibia client of the modified 7.72 tibia client. Login to your character, log-out. Then close the loader. Open blackd proxy, press 'load for tibiaclient 7.72', then let everything load. Wait around 10 seconds, log on. If It says incorrect password/username. Then you have not waited long enough and should retry again shortly(this error is caused because blackd closed the server connection as it did not load everything perfectly yet). If it comes up with a red message on your screen saying something about 'blackd', everything has worked correctly.

ALSO DO NOT RUN BLACKD OR TIBIA AS ADMINISTRATOR OR IT WILL NOT WORK!!
