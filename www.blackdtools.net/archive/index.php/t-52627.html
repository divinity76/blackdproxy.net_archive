Greetings!

This is yet another of my scripts, I've tested it for a while now so it should be fine!

What it does?
This script will hunt in the troll cave near the dwarf bridge.
When full capacity it will go to Venore to deposit the money and then continue hunting.
It will only loot money because this script was made for low level characters with few capacity.

Experience per hour?
Not hunting in the minotaur room: ~7,100

How to run it?
Go to Venore's depot (level -1), open all backpacks and hit start!
Please remember to have a shovel and a rope in your backpack.
It is set not to go into the room where the minotaur mage is but you can make it go hunt there as well just by following the next steps. (Please note that the minotaur can be lured at anytime)

Change the next line:

fastExiva __killmage = 0

with the following line:

fastExiva __killmage = 1

Feedback
Please post how many experience points per hour you get so people can see how efficient it can be.
All further feedback is welcomed, including ways to make it better.
If any errors, they will be fixed as soon as reported.
