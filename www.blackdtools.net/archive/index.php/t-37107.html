hi there, this is my second post.

I was using the Events part of BlackD Proxy for returning certain messages, which isnt so good because ppl can spam you like, hi hi hi hi hi hi and you always return the same answer.

Then I thought of this:

Go to your "Randline" folder in blackdproxy root. Make several .txt's that has one answer per line. E.g:


hi.txt contains

hi
hi there
hello
yo m8


and what you do now, is go to Events in blackd proxy and do a new event. MESSAGE THAT CONTAINS in "say".

action should now be: $randomlineof:hi.txt$

So if someone says hi, you either answer "hi", "hi there", "hello" or "yo m8".

Just a little beginning guide for you guys to evolve into an advanced Auto Responder ;)

In the rar i've put my events folder and randline folder. Just extract it in ur BlackDproxy folder and open blackd proxy, go events and load the event .txt called ev_example.txt
then it should all work ;)
