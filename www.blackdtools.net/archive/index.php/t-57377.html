This will automatically make your character ride your mount after coming out of PZ. Will not re-mount if idsmounted through other means (tough I can't think of one).
Works well with cavebot scripts.

$statusbit:10$
#number=#
1
0
0
0
exiva _mount = 0
$statusbit:10$
#number=#
0
0
0
0
exiva _mount = $numericalexp:{$_mount$}+1$
$_mount$
#number=#
1
0
1000
0
exiva > D4 01
$_mount$
#number>=#
500
0
0
0
exiva _mount = 2
