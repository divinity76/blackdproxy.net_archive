This is how I do it:

1.) Open BlackD Proxy.
2.) Open Proxy Section.
2.1.) [Uncheck] Change all Tibia clients memory so they can login to proxy
2.2.) [Uncheck] Change character list packets (use proxy for game connection)
2.3.) [Uncheck] Force login server: ~
3.) Open Tibia client.
4.1.) [Check] Forward to true servers
4.2.) [Check] Change all Tibia clients memory so they can login to proxy
4.3.) [Check] Change character list packets (use proxy for game connection)
5.) Login.

Note: If you want to multi client, then start again at step 2.), because you have to uncheck everything before opening another client and changing its IP. But lately it was working even, if you haven't done that and just opened new client and then logging in. Eventually you'll figure the system out. :p

I hope I've helped! ^___^ If this isn't working for you, post again! D:
