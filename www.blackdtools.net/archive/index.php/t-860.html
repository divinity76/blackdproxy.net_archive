The mage bomb, Wow it took me some time to realise how to work this but wow.

-- Note I did not test this with actual targets and sds just tested the logging in --

Ok, Firstly check the box in the red square.


http://i14.tinypic.com/446aut3.jpg
(http://i14.tinypic.com/446aut3.jpg if you can't see pic)

Ok now make sure you have no accounts connected and log in using your acc/password (easiest to do these one by one) Take them with whatever supplies you need etc (sd's duh) and log them out in the EXACT square you want them. Once you got all those done it's best you turn the option in the red square off.

http://i13.tinypic.com/2d8jig5.jpg
(http://i13.tinypic.com/2d8jig5.jpg if you can't see pic)

So now you have all your bombers logged out where you want them etc, Now is to set up the word part of the bomb. Write your targets name in the red rectangled area, Then choose the MS you want your bombers to keep trying to SD the target in the purple area (so if you wanted them to try sd for 10 seconds you would have 10000).

http://i13.tinypic.com/30wkok3.jpg
(http://i13.tinypic.com/30wkok3.jpg if you can't see pic)

Ok now, Press the reload button (loads up all the chars) and then select from the drop down menu(blue rectangle) a character you want in the bomber squad and click the yellow rectangled button. This adds the character to the bomber squad list(where i've written bomber one, two, three)

After this you are READY get on another char and wait/lure the target to your bomber zone when ready 'Exiva bomb' And watch the bastard die with huge success.

If your planning this ahead of time (maybe a week or so in advance) then you can use the save button so you can reload the bombers you want the target you want and shoot without having to redo it everytime you start blackd.
(probably best to use specific names to save name as aka killtom.txt Dragoondead.txt)

Hope this helps, I have NOT tested on real targets YET so these may be wrong but my knowledge of things believes me to think this is right. Have fun bombing. Blackd correct things if they need to be.

PS. If anyone has a free acc mage inferna level 18+ and doesn't want it feel free to PM the acc/pass (was worth a try :p)

PLEASE POST IF YOU NEED HELP.

Yours, Tom.
