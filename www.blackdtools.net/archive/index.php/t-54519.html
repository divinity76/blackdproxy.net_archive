Blackd Proxy 19.4
----------------------------
- Fixed a very rare bug that happened with big stack of persons in same square

Blackd Proxy 19.3
----------------------------
- fastExiva should now accept exiva check so you can use fastExiva check x,y,z
- saymessage and sayintrade can now include the comma symbol (,)
- New variable! $lastchecktileid$ will return last checked tileid
- fixed a slight path bug with scripts that repeat the exact waypoint in different lines of the script

Blackd Proxy 19.2
----------------------------
- Updated the command exiva sell:XX XX:N Packet format needed an update to be sure that it won't be automatically detected as cheat
- Be aware that now exiva sell will also count with equipped items!

Blackd Proxy 19.1
----------------------------
- Cavebot is now able to use ramps at Port Hope
- Bot is now able to autoeat all kind of food
- Fixed client freeze after some kinds of logout
- Updated tibia url in our broadcaster menu.
- New command to do game inspect in a square near you: exiva check x,y,z
- New variable to check last game inspect result: $lastcheckresult$
