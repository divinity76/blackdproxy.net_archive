The diagram.
1) This is where you enter the key that you want to use for the hotkey.
2) This is where the second key that you want for the hotkey can be entered, where it will be a combination of two keys, the key from Number 1 and 2.
3) This is where you put what you want to say when you press the hotkey
4) This will delete all the current hotkeys that you have stored.
5) This will delete the hotkey that you currently have selected.
6) This button will cancel the changes that you have done since you lasted saved the file.
7) This will save the current hotkeys to Hotkeys.ini - which will be loaded upon BlackD proxy startup.
8) Activate the hotkeys, unchecking this box will disable the hotkeys.
9) Having the box checked will repeat the selected hotkey every X seconds. X is defined in number 10.
10) How many seconds you want to pass before a hotkey is repeated

http://img422.imageshack.us/img422/5646/hotkeysyg1.jpg (http://imageshack.us)

Firstly, click on the box shown as number one on the diagram

It should then look like this;

http://img521.imageshack.us/img521/8858/keysbu5.jpg (http://imageshack.us)

Simply press the key you want to bind that hotkey too, for example if I press the end key, it will look like this;
http://img451.imageshack.us/img451/8995/keyendic0.jpg (http://imageshack.us)

Then click on the box shown as number 3 in the diagram. Type whatever you want you to say when that key is pressed, for example Exiva pause.
