How to create custom warning-sounds.


First off, you will have to download this program (http://www.dbpoweramp.com/dmc.htm) and install it.
Once you've installed it, locate the file you want to use as your custom sound. Right-click on the file and select "Convert To".
A new window will pop up. Select the right settings according to the picture:

http://img411.imageshack.us/img411/329/convertvr5.jpg
Once you've selected the right settings, just press "Convert >>".
Now you will have to go to your "BlackD Proxy"-folder and locate the sound you want to replace.
Backup the original file.
Now go back to the converted file and replace its name with the file you just backupped. (If you backupped "danger.wav" then you should name the converted file to "Danger.wav")
Copy your new "Danger.wav" to your "BlackD Proxy"-folder. You are ready to try it out!


Note: Exiva Cancel won't always stop the sound. Let's say you have a song which is 2:00 long. The alarm starts and you write "Exiva Cancel" 10 seconds after the song started to play. The sound won't stop until it has reached its end (1:50 remaining).
