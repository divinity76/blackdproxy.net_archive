IMPORTANT: In Blackd Proxy 9.373 and higher versions, the changes should be done in settings.ini

By default, Blackd Proxy allows up to 5 Tibia clients.
You can change this value:


MAXCLIENTS=5


if you set a lower value, Blackd Proxy will use less memory and CPU.
If you set a higher value, Blackd Proxy will use more memory and CPU.

Like the other ini settings, you should change this while Blackd Proxy is closed. Changes in ini files made while Blackd Proxy is being executed won't apply and will be reversed after blackd proxy closes!
