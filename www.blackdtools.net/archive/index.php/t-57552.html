Happens to me all the time when playing an OT where character flies through sqms like a rocket! (Haven't got that high character in 'rl' tibia so it probably never happend to me elsewhere other than on an OT)

Anyway, I think I've seen a slight solution for this somewhere on here, it went like this:

1.) Options -> General -> Auto Chase Off (checked)

2.) In client itself make sure you disable 'chase opponent' next to fighting mode (offensive/balanced/defensive), make sure you got it 'stand while fighting'.

Try this two, I've tried it and it kind of reduces stopping when botting on an OT, in fact now it happens very rarely. \o/
