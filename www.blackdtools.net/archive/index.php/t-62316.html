Click the goo.gl link in the post above. This will update your bdp.
If you are having trouble connecting to OTs, Follow these steps.
1. Get fresh with BDP!, click the advanced button in bdp, and reload the config for a different version
2. Make sure you set the version of the OT client when you re-open bdp.
3. Check your path to the client you want to use.
4. Ensure your automap is in the correct place. OT servers often have custom maps. If you use the default, all maps on the OT server will overwrite, and cause errors with your original automap. Unless you are playing on a server with a real map.
5. Finally, make sure you have the proper OT server IP in place, as well as the login port (7171) by default for most servers unless otherwise noted.
6. Start your game.

I play on tibian a lot. Its an OT server with a real map, just boosted exp rates and free premium for everyone.
Anyone that wants to play in a server with a custom client, will probably have to change the client name. For instance Tibians custom client, is Tibian.exe. Blackd will not find Tibian.exe in the folder i selected, because its not named Tibia.exe. Simply change the name, and that should be fixed.

These are all the troubleshooting steps you should probably need.
