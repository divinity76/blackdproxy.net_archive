01-01-2012, 20:33
I think it is only a small warning because I left my config instead blank config in last update.

manual fix:

Go to blackd proxy folder and edit settings init, delete the value of TibiaPath so you have only this:

TibiaPath=

(or directly write the correct value if you know it)

In config942\config.ini , edit the value of TibiaExePath so you have only this:

TibiaExePath=

(or directly write the correct value if you know it)
