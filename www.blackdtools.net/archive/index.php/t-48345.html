1. If you already have a Blackd License download it from http://www.blackdtools.net/updates.php

2. Install it, use any of the installers. For vista/7 I guess the alternative installer will work better

3. The main file of BSC is called Tibia.exe (it is called this way to add stealth power) Execute it directly or just click on the link that the installer should have placed.

4. Open the cheats in the menu "Open Cheats"

5. Light should be enabled (disabled by default) and HP/mana should be configured. You should also define the hotkeys in Tibia! (make hotkeys that use potions on yourself)

6. Tibia client should be focused or cheats will not work!


Note:We have not tested it in Windows 7, but if anyone got an error message please let me know and I will try to fix it asap.
