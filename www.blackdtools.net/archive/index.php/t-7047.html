First of all, just to say this tutorial is about how to add your old map files, to new Tibia system so you can use it.

1. Go to Start > Settings > Control Panel and open Folder Options.
You should get something like this:
http://www.winmobiletech.com/kuvat/WinFileExplorerShowHiddenFiles.png
then you just mark Show hidden files and folders.

When we did that all you can close all.

2. Go to My Computer > Local Disk (on witch is windows) > Documents and Settings > "your user" (default Administrator) > Application Data > Tibia > Automap

When you open folder Tibia there will be Tibia.cfg (hotkeys and all settings) and folder Automaps, open Automaps and past your maps in it.

Then start Tibia, login and start playing ;)
Hope it helps.
