http://i2.photobucket.com/albums/y9/alucarr/bptutorial/trainer881.jpg

1. Attack the mother slime for a second, then stop fast.
2. Press button [10] (Get ID of last attacked)
3. Enable checkbox [9] (Avoid attacking the monster with this ID)
4. Go to cavebot and load slimetrainer.txt
5. Enable cavebot (not shown in the current old picture, but you will find that option in colour green near point [1] )

IMPORTANT NOTE: if you are in a corridor then avoid that targeted slime go between another creature and you. Else cavebot will think that reaching target is impossible and it will stop attacking.
