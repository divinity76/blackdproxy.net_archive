I don't understand why blackd proxy would try to find tibia.dat in windows/system32 . That is really strange.
Maybe you installed blackd proxy in temp folder or there is a big mess with your Windows.
Solution:
1. uninstall it.
2. download latest installer
3. run installer as admin. Ensure you install it in default folder (something like C:\program files\blackd tools\blackd proxy)
4. run blackdproxy and tibia. If it doesnt work then close both and run both as admin. If it doesnt work (and if you have windows vista or windows 7) then go to your windows control panel / user accounts , select your account and somewhere there select that you want the minimum level of security. Then restart computer and reinstall blackd proxy again. It should work now, no doubt. Test it. After that you can go back to
control panel / user accounts and restore your previous security level.
