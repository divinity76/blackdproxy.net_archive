IMPORTANT: In Blackd Proxy 9.373 and higher versions, the changes should be done in settings.ini

To test what happens with gms you can edit the ini and redefine the line where it says





GmStartWithThis3=gm -





For example, if you want that people starting with "bla" gets considered as gm it would be:



GmStartWithThis3=bla-





Then for example Blackd Neo would be considered as gm.



IMPORTANT: while this is redefined the real gms will be considered as players. (except a few gms with special names not starting with "gm", wich are always considered gms, for example: Guido)
