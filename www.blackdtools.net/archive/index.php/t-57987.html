With current version of Blackd Proxy you can make a simple script for autolooting all:



setBot LootAll=1
setLootDistance 1
gotoScriptLine 0


Important note: It is a very simple script and I think that it is not detectable, but I won't guarantee that yet.

note2: it will give alarm because you are not adding any creature to be killed. It will think that a pk is attacking you and it will give alarm with any monster. I will try to make a new cavebot command to disable this alarm in the next version...
