Script tested at Tibia 8.71 & BlackD Proxy 19.0

This simple script allows you to exp on the dwarfs in the kazz city. You don`t need to go out of kazz. Just go to the place marked at the screen, and run the script. That`s all !

http://blackdtools.net/cavebotimages/gex_000000007.jpg

Looting gold is enabled. Script is going down the floors, so you need to get a shovel, rope, and some of food, etc, and should have configured autohealer.
Monsters that it kill: Dwarfs, Dwarfs Soldiers, Rotworms.

note: if any bug or request to this script is needed just send PM to GeXem and he will answer you soon as possible.
