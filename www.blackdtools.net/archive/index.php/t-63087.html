Good news:
From today all my Tibia 11 tools are open source and 100% free for everybody! (no need to purchase anything anymore).

Bad news:
You can expect all known cheat tools to stop working after the incoming big Tibia patch of 21 March 2017 (they will include a very aggresive cheat detection spyware that can read everything in your computer).

What can you expect after 21 March 2017?
- Nobody will be able to cheat using PUBLIC software. Of course some people might be able to keep doing small cheats using custom private small secret bots.
- Forget about cavebots. After a few reports they will detect what software is causing the movement and then they will blacklist that software.
- If you are an experienced programmer and you like cheating, then you might be able to build a working private bot only for you and maybe for a pair of friends. However it would go blacklisted really fast in case you publish it and/or a lot of people start using it.
- I probably won't waste my time trying to make a working public cheat for official Tibia. I will only keep selling gold and codes.

Can we bypass that new detection system?
- Impossible is nothing. However it will be EXTREMELY HARD. You should keep in mind that they have a lot of expert people working on it every day and releasing updates to fix any possible way of bypass.
- We should fight a public anticheat system: BattlEye. A lot of programmers of many important games are already thinking a working method, but this is not exactly easy. BattleEye is a strong spyware that can know your running proccess, window titles, installed hardware, installed software and a lot more.
- Possible solution 1: Make a custom Tibia client without BattleEye.
- Possible solution 2: Patch/Inject dll into Tibia client to bypass BattleEye and bypass its detection for BattleEye presence.
- Possible solution 3: Build a special system enviroment where Tibia.exe can run without real system information. Tibia.exe should obtain standard results not containing your cheat process/folder/files. Maybe you can run Tibia.exe inside a virtual machine and you can run cheats outside that virtual machine. If the cheat software running outside is able to read/write memory of Tibia.exe then it can make some undetectable magic.
