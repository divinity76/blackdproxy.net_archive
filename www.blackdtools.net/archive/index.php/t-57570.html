Btw, the main problem is Cipsoft changed the logout message so bot can't detect it right now.

Old message: You have been idle for 15 minutes. You will be disconnected in one minute if you are still idle then.

New message: There was no variation in your behaviour for 15 minutes. You will be disconnected in one minute if there is no change in your actions until then.

blackd proxy 22.7 will consider this fact and will also allow you to customize that server logout message for correct detection of the autologout event in ot servers. Anyways -by default- all older tibia versions will be loaded with older logout message, and all newest tibia versions will be loaded with new logout message.
