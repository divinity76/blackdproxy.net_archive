The procedure to find addresses in Tibia 11:
1. Search what you want while your player is playing, using the old procedure. That is: you start a search with an known value (faster) or with unknown value, then you change something in your player, you search next, you change something, you search next... until only 1 value is remaining. You obtain a final address.
2. Do pointer scan over that obtained final address. You should obtain a huge list of possible paths. Don't close this list.
Best settings for most cases:
https://blackdtools.net/cheat_engine_tibia11.jpg

3. Close Tibia 11, open it again, and login a player again.
4. Reattach Tibia 11 to Game Cheat, and repeat only (1) so you obtain a new (different) final address.
5. Go back to the obtained list in (2) and go Pointer scanner>Rescan memory. Enter new final address. Enable Repeat rescan until stopped. Press ok to start filtering.
6. Walk a bit,change floor and do some actions with your player. Around 15 seconds should be enough to filter a lot of address.
7. Go back to the list and press "Stop rescan loop". Now you should have less address.
case 7a. If they are still a lot then repeat everything from point (3)
case 7b. If the list is small (<20 address) then test the shortest paths (less steps) that start at a .dll because they are usally the right ones. In some cases you always get a few valid path solutions.
8. Test the obtained path solutions in a different computer with a different windows version, in order to discard paths that depends on operative system.

You can find the address for the current session using old tactics.
However, that address will only be valid in your machine until you close Tibia.
Nowadays your goal is obtaining universal paths to obtain that dynamic address so you need to do the procedure several times [open tibia, find current address, close tibia]
until you only havew few paths remaining. Such paths should work for any future session of Tibia, in any machine.


And finally, answering to your last question: No, there is not a single short address in tibia 11 like that. Everything are paths now.
And in the case of the helmet, I could bet it is now part of a collection of equipment slots. You will need to search that collection start. However, it should be a static collection, so obtaining a path there is still possible. But you can expect it to be a very long path (maybe with 9, 10 or 11 steps)
