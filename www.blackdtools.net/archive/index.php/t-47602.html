Greetings!

This is my third script, I've tested it for a while now so it should be fine! The only part that I haven't tested it the running to depot on danger.

What it does?
This script will hunt in all the floors of Liberty Bay Vandura Cave.
When full capacity it will go to town sell the maces, swords, tarantula egg, steel helmet and centipede legs, deposit the money and refill manas and health potions to then continue hunting.

How does it work?
This script was made for personal use, so beware of the settings. It is set to buy 10 mana potions and 10 health potions. If at any point it has less than 5 potions of any kind it will go sell stuff, deposit money and refill. It will always keep a steel helmet (that is because my character uses one and I didn't want it to be sold).

What does it kill?
Bugs
Wasps
Skunks
Rats
Rotworms
Carrion Rotworms
Spiders
Poison Spiders
Centipedes
Tarantulas


How do I use it?
I have a knight of level 25ish with skills 50/50 (Knight set, Steel Helmet, Serpent Sword and Dragon shield) and it almost never uses the potions.
I set Blackd to use 'Exura' when hitpoints below 80%, a health potion when below hitpoints below 30% and a mana potion when mana below 30%.
I use Runemaker to set auto eat and to say 'Utevo Lux' when mana is higher than 100.

Where should I place my character in the beginning?
In Liberty Bay depot or near.

Warning
It can get tough for low level with bad skills or bad equipment due to the tarantulas.

Feedback
All feedback is welcomed, including ways to make it better.
If any errors, they will be fixed as soon as reported.
