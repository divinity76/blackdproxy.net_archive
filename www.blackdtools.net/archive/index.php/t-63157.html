SHORT BACKGROUND​

Because so many players are complaining about new Tibia versions and they want Cip~
to create a legacy 7.4-8.0 official server, we decided to build the most similar server to the Real Tibia from version 7.6.

Based on leaked files we've converted everything into XML, LUA and OTBM and made sure the scripts are stable and working correctly. Now we have 100% Real Map, Real NPCs, Real Monsters and Real Quests. Only spell formulas might be slightly different, as they were hardcoded in the engine and the true formulas are still unknown.

This is a low-rate server as we want to keep it online for the years. In a future, there's a plan of adding new high-rate world and worlds with 7.4 and 8.0 versions as well!

We all see these oldschool servers, which are actually the 8.0+ with the sprites replaced. Chrona is a true 7.6 server so you can use an official Tibia 7.6 client with an IP changer! Because the basic client doesn't have any antibot mechanism, we're in a process of implementing server-side bot detection tool based on a player behaviour.
