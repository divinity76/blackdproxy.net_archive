Script tested with Tibia 8.71 & BlackD Proxy 19.1

Another script for dwarfs in kazz. But this one allows you to exp on the main cave (the biggest) in kazz (out of the city).

See at the screenshot where to activate it:

http://blackdtools.net/cavebotimages/script46.jpg

Script is going down the floors, so you need to get a shovel, rope, and some of food, etc, and should have configured autohealer.
Monsters that it kill: Dwarfs, Dwarfs Soldiers, Rotworms. If any bug or request to this script is needed - just send me a PM. I`ll answer soon as possible.
NOTE: I made this script for low levels, so script is not going to down floors, where are dwarfs guards.
