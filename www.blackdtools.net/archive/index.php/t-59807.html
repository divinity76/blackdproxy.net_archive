In the section "Runemaker" check the box that says "Play player.wav when an unfriendly player-creature pop on screen". Make sure this box is checked, if the player.wav isnt loud enough for you, get a new sound, make sure it is a .wav file call it "player.wav" now you can have big alarms when people show up, and easy enough to add in "exceptions" in the top right hand corner of the "Runemaker" you can see it say "Dont consider following names as Danger" look down a little bit and you can see the box "Add exception" Type in a creature you may be training with ex Rat? or a Friend who is Mana training beside you ex conteh.

Hope this helped at all, this is mainly for mana training/training AFK and able to listen for alarms. If you are running the "Cavebot" whenever you are attacked the alarms will automatically go off.
