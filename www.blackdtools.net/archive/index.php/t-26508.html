Here I sum up what a good cheater should know to avoid ban:

1. Always have speakers on while you cheat, so you can listen the gm alarms.

2. If a gm popup on you. You should go to cheats menu and you should DISABLE the checkbox on the top: [ ] "Activate hardcore cheats" From now you can type anything and any command and you will act as if you were not using any cheat at all. After the gm is gone you can reenable that checkbox again and reload blackd proxy. Ensure that the gm is no longer online before resume your cheating!

3. Never reconnect with cheats during server save. You will need to do a complete new login after server save. Using the old login packet might give you an automatic ban.

4. Don't modify your file tibia.exe . It probably can be detected by now. If you had already modified it then uninstall and reinstall tibia again. Both LoW and me are giving a new and free multiclient that doesnt need to modify tibia.exe

5. Don't connect same mc characters very fast every day. If every day they see that char X and char Y connects with same IP in less than 10 seconds then you can be banished by login logs. Wait around 1 minute before connecting your next mc.

6. If tibia client debugs/crashes : In this case you should uninstall tibia and install it again to ensure that it doesnt autosend debug next time you open tibia. Maybe deleting the error file would be enough, but better don't risk and reinstall tibia completely. Backup your maps first!
