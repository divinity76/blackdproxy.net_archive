Greetings!

I hope you enjoy this script, it will bring a character to level 8 way faster than my non premium script.

What it does?
This script will take a new character and level it until it has 600 experience points. It is fully automated and no interaction is needed between the character and the person.

How does it do it?
This script will make the character hunt for food to make cash in order to buy decent equipment before starting to hunt. This is done in the FACC side.
After it reaches the specific amount of golden coins it buys all the equipment and then begins hunting in the sewer cave from level 1 through 4. When it reaches level 4 it then goes hunting on the Western part of the island since it is faster than hunting rats until level 8.
It also has an auto-continuer in case you get kicked or logged out. (See instructions for details).

What does it kill?
For Money: Pig
Deer
Sheep
Rabbit

Levels 1-4: Rats
Cave Rats

Levels 4-8: Spider
Poison Spider
Wolves
Deer
Bugs
Snakes


What equipment does it buy in rook? Backpack
Sabre
Leather Armor
Leather Legs
Wooden Shield
Studded Helmet


Other Comments

This script was designed so it can hunt in the area where the pigs and sheep are. Do not worry about it getting trapped with the gates as it will not happen.
It was decided that the wolves cave will be avoided since while testing it was found that even characters of level 7 or 8 can die when sorrounded by 3-5 wolves.
Some part of the script where taken from Arne's Script (http://blackdtools.net/showthread.php?t=34020), but everthing in the script was tested carefully. Thanks Arne!
It was noted that although it will get experience faster the character will not loot as many golden coins as other scripts where it just hunts rats but the purpose of this script was to get it to the mainland fast, after all 600 gps are easier to get there.
Tip: The script was successfully tested while using balanced fighting.


Instructions
1. Create a new character.
2. Login in Tibia.
3. Start script.
Note:Do touch anything else; it will do everything for you. Just make sure Blackd is running actively in the client.
Remember: Uncheck allow reposition at start.

Instructions for Re-running the Script
This are instructions for when you need to re-run the script on a character that for some reason stopped in the middle of the script running (kick, etc.)
1. Open Tibia and Log in.
2. Bring your character to the temple.
3. Start the script.
Warning: Do not open the bag/backpack.

Feedback
All feedback is welcomed, including way to make it better.
It has been tested with many characters and is still under testing (although there shouldn't be any errors). If any errors, they will be fixed as soon as reported.
