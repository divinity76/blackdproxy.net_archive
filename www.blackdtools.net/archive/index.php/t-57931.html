you can make a small script, set what you want to kill and then end the script with gotoscriptline 0
Then just enable that script while you are hunting.

Since Blackd Proxy 23.7 you can also loot everything if you add this in the start of the script:
setBot LootAll=1


In this new loot mode Blackd Proxy will loot everything by default. It is a new inverse loot mode. You can add exceptions with the old command setLoot XX XX
If you add food ids then it will not loot them. It will eat them instead.
And you might also want to set backpacks and bag ids as exceptions in this mode (else you will loot a lot of backpacks and bags)
