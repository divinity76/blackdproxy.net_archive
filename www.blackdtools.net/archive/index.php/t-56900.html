You can modify the config.ini file for the version 8.57 and change the ID of the mana potions and replace it for the ID if your mana rnues. Something like this

Go to this folder and look for the file config.ini

C:\Program Files\Blackd Tools\Blackd Proxy\config857

Open the file and look for this very specific line

tileID_mana_potion=0C 01

and change it to this

tileID_mana_potion=62 0C

After that, just set the bot to refill your mana with a mana potion in the HP & MANA menu, and it should work.
