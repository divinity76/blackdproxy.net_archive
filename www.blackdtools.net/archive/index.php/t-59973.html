1.) Open BlackD Proxy.
2.) Click on 'Advanced' section.
3.) In Advanced section search for 'CLICK HERE TO load a config for a different Tibia Version. This will close ALL. Reload later.' Click on it.
4.) BlackD Proxy will close itself.
5.) Open BlackD Proxy again, this time start up window will appear.
6.) Choose Tibia version, Tibia client path, Tibia maps path and perhaps IP of the OT server you're trying to connect.
7.) Click 'CONTINUE'.,
8.) Enjoy! :)
