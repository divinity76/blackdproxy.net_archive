First of all you should understand that Tibia is connecting to server through Blackd Proxy.
If Blackd Proxy does not have enough CPU to process the packets then it won't be able to deliver the packets in time = it will look like having some more lag. In some cases you will also lose connection!

If you are reading this forum then your CPU is good enough for Blackd Proxy! What is the problem then?
Possible problem 1 = You are running Tibia at max frames per second -> no cpu left for Blackd Proxy
Possible problem 2 = You are running too many Tibia clients -> no cpu left for Blackd Proxy

THE SOLUTION:
- Decrease Tibia framerate in Options > Graphics > Advanced > Adjust framerate limit > set 25 or less. Ensure that the displayed framerate counter keeps stable there. If it is not stable then decrease it even more. You might need to do that in all your multiclients. More multiclients = you should set less framerate limit in all.
