This error happens because somehow your tibia.dat is outdated. Somehow you didn't update Tibia correctly , or you installed the update somewhere else. Blackd Proxy expects to find an updated Tibia at
C:\Program Files (x86)\Tibia\Tibia.dat

Try executing C:\Program Files (x86)\Tibia\Tibia.exe
(without any cheats running)
If something is not update then it should update now.

After that just try Blackd Proxy again.
