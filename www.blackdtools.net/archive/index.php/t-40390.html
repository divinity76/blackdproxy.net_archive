serababy, you can use Blackd in more than 5 accounts at the same time, but you have to edit a file to do so.

Go to ..\Blackd Proxy\config850\ folder and open the file config.ini using a text editor like notepad.

Edit the line "MAXCLIENTS=5" to "MAXCLIENTS=9" if you want run 9 accounts at the same time for example.

Note you have to do it while your Blackd is NOT open, otherwise your configurations will not be used.
