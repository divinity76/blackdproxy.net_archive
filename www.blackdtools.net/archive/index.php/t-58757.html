If you use conds to heal you can use this command:


exiva drop XX XX AA

Where XX XX is item ID and AA is the amount you want to drop under you.

As he already stated, if you are running a cavebot script, put this at the end before gotoscriptline 0 and write exiva drop xx xx aa

xx xx = item id of empty potion bottles

aa = quantity of potions to drop

Also, if this is at the end of a script it may have to look like ; sayInTrade exiva drop XX XX AA
