If the server uses the same "command" to use an item like RL Tibia, then it should work (most OTs change that to make botting difficult). I'm assuming the item you need to use is in your backpack.




$myhppercent$
#number<#
90
500
1000
exiva > 82 FF FF 00 00 00 37 17 00 00


It will check every 500ms (half second) to see if your HP is less than 90%, if it is, it will use your item. If you need to use the item again, it will wait 1000ms (1s) from the last time used to use it again. It's gonna use it as a hotkey, so no open bp needed.
